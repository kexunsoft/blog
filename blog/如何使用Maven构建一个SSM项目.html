开发环境:

  1. 开发工具 eclipse
  2. JDK版本:1.8
  3. 操作系统:win10

 

创建一个maven项目

1.打开eclipse

2.file>new>Maven Project

注意:一般第一个就是 如果没有找到就可以点击 other

 

 

3.将create a simple project.... 多选框勾上 这里是创建一个简单的maven项目 点击next(下一步)

 

group id 组织名

artifact ID 项目名

version 版本

packaging 打包方式

  • ​ jar 打包从java项目

    ​ war 打包成 web项目

    ​ pom 做聚合,或者是作为父项目

点击next 即可

 

到此项目已经创建好了 但是在 pom.xml 会有吧 ×

怎么解决呢

右键项目>properties(最后一个) 显示如下

找到 project facets

在右边找到 Dynamic web module 将复选框勾掉 点击 右下角的 apply 然后在将 其勾上 下面会出现一个黄色的提示 点击会弹出一个对话框 将distory修改成src/main/webapp

然后点击apply and close 现在就不会报错了
